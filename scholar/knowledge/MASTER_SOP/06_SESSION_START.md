# 06: Session Start (Quick-Start Checklist)
**Purpose:** Eliminate "I do not know where to begin" and start a session in minutes.

---

## 60-Second Start (Minimum)
1) Check the Progress Tracker (07): last session and next action.
2) Pick 1 LO or micro-target for today.
3) Materials ready?
   - Yes: proceed.
   - No: run Material Ingestion (05) for 15 minutes.
4) Choose mode (Core, Sprint, Drill, Light, Quick Sprint).
5) Begin Entry (M1) and start learning.

---

## Full Start Checklist
### 1) State Check
- Focus level (1-10)
- Energy/motivation

### 2) Scope Check
- Topic or LO for today
- Time available

### 3) Materials + Source-Lock
- List exact sources (pages, slides, timestamps)
- If missing, run Material Ingestion (05)

### 4) Mode Selection
- New material -> Core
- Review/test -> Sprint
- Short session -> Light or Quick Sprint
- Repeated misses -> Drill

### 5) Plan of Attack (3-5 steps)
- Example: Prime -> Encode 1 bucket -> Quick retrieval -> Wrap

### 6) Pre-test or Brain Dump
- 1-3 questions or 60-120 seconds of free recall

---

## If You Are Resuming
- Paste your last session log or summary
- Read your Progress Tracker row
- Start with the "Next action" listed there

- Source-lock confirmed
