# Legacy Versions

This directory consolidates archived PT Study SOP releases for reference.

- **Shared references:** audit notes, changelog, methods index, master plan, usage/server helpers, and version reference materials.
- **v7:** concise/full SOP 7.1 variants, single-session v7.3, v7.3+v7.4 markdown, and the v7 core SOP.
- **v8.1:** archived concept and systems review, master index, troubleshooting guide, and meta revision log.
- **v8.4_Tutor_Edition:** tutor-oriented SOP modules plus runtime prompt.
- **v8.5.1_pre_safe_deploy:** pre-safe-deploy SOP modules, framework library, and prompts.
- **v8.5.1_Spotter_Prime:** Spotter Prime bundle with core docs and nested v8 historical releases (v8.1.1, v8.3, v8.4, v8.5.1).
