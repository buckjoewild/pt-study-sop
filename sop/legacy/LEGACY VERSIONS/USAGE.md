# PT Study SOP v8.3 - Usage Guide
## Quick Start
Copy the text from `releases/v8/PT_Study_SOP_v8.3/Runtime_Prompt.md`.

Paste it into a new ChatGPT session.

Command: "Resume [Topic]" or "Let's Study [Topic]".

## The 3 Modes (Triage)
The system will auto-select one of these three gears:

1. SPRINT MODE (The "Crunch" Gear)
Best for: Exams tomorrow, huge topics, need speed.

What happens: Surface scan + Memory Hooks. No deep mechanisms.

Output: List of terms with vivid hooks.

2. CORE MODE (The Standard)
Best for: Learning a topic for the first time.

What happens:

MAP: AI selects framework & anchors (Silver Platter).

LOOP: You learn 3 anchors -> Retrieve immediately -> Connect them.

Output: Deep understanding + Anki Cards.

3. DRILL MODE (The Review)
Best for: Keeping old topics fresh.

What happens: Zero teaching. Rapid-fire questions only.

Output: Weak point list.

## Key Commands
"menu" -> Show the HUD (Phase, Mode, Hook Style).

"Resume [Topic]" -> Restores context from your last recap.

"Sprint" / "Deep" -> Forces the mode change immediately.
